## [1.0.0] - 2026-04-18

### Changed
- “专注详情”的默认粒度从“周”调整为“日”，更适合查看近期趋势。

## [0.1.1] - 2026-04-18

### Added
- 新增设置页与统计相关视图，支持查看专注数据并展示专注时长。
- 新增分类标签与日维度统计能力。

### Changed
- 默认关闭自动跳过番茄；暂停时支持跳过当前阶段。
- 统一窗口尺寸与界面样式，减少不同面板之间的跳动感。

### Fixed
- 修复开始倒计时后按钮状态更新延迟的问题。
- 修复暂停状态下仍可输入或出现异常的多个问题。
