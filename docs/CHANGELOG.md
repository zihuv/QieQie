## [1.0.2] - 2026-04-20

### Changed
- 将分类标签目录从偏好设置迁移到本地数据库统一管理，减少标签与历史记录分散存储带来的不一致。

### Fixed
- 启动时会自动规范化旧版专注记录，并迁移历史分类数据，避免升级后标签或说明丢失。
- 修复进入休息阶段后必须先开始休息才能跳过的问题，未开始的短休息和长休息现在也可直接跳过。
- 修复短休息结束返回下一轮专注时，菜单栏偶发退回初始时钟图标而不是显示待开始倒计时的问题。

## [1.0.1] - 2026-04-20

### Changed
- 调整主弹窗中“分类与说明”行的分类选择器宽度，让说明输入框获得更多可用空间。

### Fixed
- 修复跳过进行中的专注时不写入已专注时长的问题。

## [1.0.0] - 2026-04-18

### Changed
- “专注详情”的默认粒度从“周”调整为“日”，更适合查看近期趋势。

## [0.1.1] - 2026-04-18

### Added
- 新增设置页与统计相关视图，支持查看专注数据并展示专注时长。
- 新增分类标签与日维度统计能力。

### Changed
- 默认关闭自动跳过番茄；暂停时支持跳过当前阶段。
- 统一窗口尺寸与界面样式，减少不同面板之间的跳动感。

### Fixed
- 修复开始倒计时后按钮状态更新延迟的问题。
- 修复暂停状态下仍可输入或出现异常的多个问题。

## [0.1.0] - 2026-04-12

### Added
- 首个公开版本。
